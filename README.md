# Sliding Puzzle
_Current version:0.1_

Small sliding puzzle game built with jQuery. Made just for fun!
Try demo [here](http://sliding-puzzle.olefrank.dk)

## Compatibility
Tested OK in 
* Google Chrome Desktop/Android
* Android 4.3 browser
* IE 9+
* FireFox 27+
